The Double-Twist Ice-Resistant Net Revolution

Faced with freezing storms and broken nets that even the elder seals mocked, I invented the Double-Twist Ice-Resistant Net. Made from woven narwhal hair and blessed eel slime (for flexibility), this net withstood the coldest gales. Thanks to this marvel, our village increased winter fish hauls by 30% - and the seals stopped laughing. I even taught a workshop called "Weaving Without Weeping" to help the next generation master the art.

This website was developed as part of the second Capstone Project challenge, Online Resume, in Angela Yu's Full-Stack Web Development Course on Udemy. The content features the online resume of a fictional character from the World of Warcraft universe. No real personal data has been used, ensuring privacy, as this project is publicly accessible on the internet. This time it's not only HTML but CSS as well.

Project Description: The goal of this project is to create a personal website that showcases your skills, experiences, and interests. The website should be visually appealing and easy to navigate, while also providing relevant information about you and your background. The site should be built entirely with HTML and CSS, with no JavaScript code included.
